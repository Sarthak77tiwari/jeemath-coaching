Real JEE Questions?

These are actual Class 11 problems that appeared in JEE Mains. If you struggle with 2 or more, this program was built for you.

Sets & Relations ⚡ Medium
Let A = {1, 2, 3, 4} and R = {(a, b) : b = a + 1}. Is R reflexive? Symmetric? Transitive? Justify each.
💡 Tip: Check each property definition precisely. For transitive, hunt for a counterexample.
Not reflexive — (1,1) ∉ R (since 1 ≠ 1+1)
Not symmetric — (1,2) ∈ R but (2,1) ∉ R
Not transitive — (1,2) ∈ R and (2,3) ∈ R but (1,3) ∉ R
Functions ⚡ Medium
Find the domain of f(x) = 1/√(x² − 5x + 6). Is f one-one? Is f onto ℝ?
💡 Tip: The expression inside √ must be strictly > 0. Factorize (x−2)(x−3) and apply sign chart.
Domain: (−∞, 2) ∪ (3, ∞)
Not one-one — symmetric values about x = 2.5 give same output
Not onto ℝ — range is (0, ∞) ⊂ ℝ
Trigonometry 🔥 Hard · JEE 2023
Prove cos 3x = 4cos³x − 3cos x. Hence find the exact value of cos 20° × cos 40° × cos 80°.
💡 Tip: Use the cos(2x+x) expansion. Then set x = 20° in the triple-product identity.
Using the identity: cos 20° · cos 40° · cos 80° = (1/4)·cos(3×20°) corrected via product formula = 1/8
This appeared in JEE Mains 2023 — most students left it blank.
P&C 🔥 Hard · Common Trap
In how many ways can 4 boys and 3 girls be seated in a row such that no two girls sit adjacent to each other?
💡 Tip: First arrange boys. Count gaps including ends. Place girls in gaps only.
Boys can be arranged: 4! = 24 ways
5 gaps created (including ends) → choose 3 gaps for girls: ⁵P₃ = 60
Total = 24 × 60 = 1440 ways
Most students forget to count end positions — and lose all 4 marks.
Sequences ⚡ Medium · Classic Trap
If Sₙ = 3n² + 5n, find Tₙ. Is T₁ consistent with S₁? What is T₇?
💡 Tip: Tₙ = Sₙ − Sₙ₋₁ for n ≥ 2. ALWAYS check n=1 separately — this is where 79% of students lose marks.
Tₙ = Sₙ − Sₙ₋₁ = 6n + 2 (for n ≥ 2)
Check: T₁ from formula = 8. S₁ = 3+5 = 8. ✓ Consistent.
T₇ = 6(7)+2 = 44
